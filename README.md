# Weight functions
Tables containing weight functions for parallel addition of numeration systems described in my Master's Thesis.

Consider the constant <img src="https://latex.codecogs.com/svg.latex?\small\rho=" title="\rho=" /> <img src="https://latex.codecogs.com/svg.latex?\scriptstyle\frac{+1+i\sqrt{3}}{2}" title="\frac{+1+i\sqrt{3}}{2}" />.

The weight functions for the following numerations systems can be seen:
- bases <img src="https://latex.codecogs.com/svg.latex?\small\pm1+i" title="\pm1 + i" /> and alphabets <img src="https://latex.codecogs.com/svg.latex?\small\{-1,0,1\}" title="\{-1,0,1\}" /> and <img src="https://latex.codecogs.com/svg.latex?\small\{-1,-i,0,1,i\}" title="\{-1,-i,0,1,i\}" />,
- base <img src="https://latex.codecogs.com/svg.latex?\small\.i\sqrt{2}" title="i\sqrt{2}" /> and alphabet <img src="https://latex.codecogs.com/svg.latex?\small\{-1,0,1\}" title="\{-1,0,1\}" />,
- base <img src="https://latex.codecogs.com/svg.latex?\small\.i\sqrt{3}" title="i\sqrt{3}" /> and alphabet <img src="https://latex.codecogs.com/svg.latex?\small\{0,1,-1,\rho,\rho^2,-\rho,-\rho^2\}" title="\{0,1,-1,\rho,\rho^2,-\rho,-\rho^2\}" />,
- bases <img src="https://latex.codecogs.com/svg.latex?\scriptstyle\frac{\pm3+i\sqrt{3}}{2}" title="\frac{\pm3+i\sqrt{3}}{2}" /> and alphabet  <img src="https://latex.codecogs.com/svg.latex?\small\{0,1,-1,\rho,\rho^2,-\rho,-\rho^2\}" title="\{0,1,-1,\rho,\rho^2,-\rho,-\rho^2\}" />,
- bases <img src="https://latex.codecogs.com/svg.latex?\scriptstyle\frac{\pm1+i\sqrt{7}}{2}" title="\frac{\pm1+i\sqrt{7}}{2}" /> and alphabet <img src="https://latex.codecogs.com/svg.latex?\small\{-1,0,1\}" title="\{-1,0,1\}" />.

Since some of the weight functions are too big, they are divided into several parts which are in the same folder. 

Most of the files are rendered directly on this page.
